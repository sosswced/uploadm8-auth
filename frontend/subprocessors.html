Privacy Terms Security Cookies Subprocessors Refunds DMCA Support
Third-party subprocessors help us run UploadM8. Core stack: API and worker on Render, PostgreSQL, Redis, Cloudflare R2, email Mailgun, payments Stripe, and AI/analysis via OpenAI. Depending on feature flags and keys, we may also call Google Cloud (Vision, Video Intelligence), Twelve Labs, ACRCloud, Replicate, fal.ai, remove.bg, Pikzels (api.pikzels.com), SerpAPI / YouTube Data API (trend intel), and Sentry (errors/tracing when configured). Public API host: auth.uploadm8.com; web app: app.uploadm8.com (see deployment BASE_URL / FRONTEND_URL). We give at least 30 days notice of material subprocessor changes where required.

Pikzels (Thumbnail Studio)

When a Pikzels API key is configured on our servers, Thumbnail Studio sends your prompts and images to Pikzels (https://api.pikzels.com) for generation, edits, scoring, titles, and persona or style helpers. Your browser stays on UploadM8; we forward the request so your key is not exposed client-side.

ProviderRoleDataPrivacy
Pikzels Thumbnail Studio proxy (API v2) Images, prompts, and parameters you submit in the Studio UI or API pikzels.com · docs
