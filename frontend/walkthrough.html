How it works

Last updated: March 16, 2026

Operating model: Upload once, distribute everywhere — with reliability, auditability, and control.

Core values

  • Consistency: predictable publishing cadence across platforms.
  • Control: you choose platforms, privacy, titles/captions, and timing.
  • Observability: transparent status, logs, alerts, and KPIs.
  • Security: OAuth-based access, least privilege, token hygiene.

How it works

  1. Connect platforms via OAuth in Connected Platforms.
  2. Upload videos and select destinations (TikTok, YouTube Shorts, Instagram Reels, Facebook Reels).
  3. Choose timing: post now, smart schedule, or manual scheduling per video.
  4. Processing pipeline runs: transcode → audio transcription via Whisper (Stage 5.5) → thumbnail generation → AI captions enriched with speech context → publish.
  5. Monitor queue countdowns, success/failure, and platform-specific results.
  6. Get notified via Discord webhook (testable from Settings).

Smart Scheduling (UI-first)

The UI supports both manual scheduling and a “Smart schedule” mode. Smart schedule becomes fully functional once the backend exposes recommended time slots per platform/account.
